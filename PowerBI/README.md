# Unveiling the 'Global Electronic Retailer Report' in Power BI.

This interactive report consolidates key performance indicators from worldwide sales data, enabling viewers to spot trends, compare periods, and run what‑if scenarios with just a few clicks.

What I brought to the project:

1)  Data cleaning & transformation in Power Query to ensure a reliable, analysis‑ready model.

2)  Custom DAX measures for sales, profit, and quantity that deliver accurate, context‑aware calculations.

3)  Geo‑visualization – converted a GeoJSON/Shapefile to TopoJSON via MapShaper and integrated it into Power BI’s Shape Map for a crisp regional view.

4)  Drill‑through Tree Map that lets users click a category and instantly see detailed customer‑level insights on a dedicated page.

5)  Created Role Based Access for US People's access & also created incremental Refresh for last 90 days data and set the last 12 months data before 90 days to be Archived.

Main takeaways :

1)  SQL Server connection.

2) Tailored visual storytelling custom charts turn complex numbers into intuitive, at‑a‑glance insights.

3) Responsive, clean layout – works flawlessly on desktop and mobile, making insights accessible wherever you are.

4) It demonstrates the value of turning raw sales data into a story that guides next steps.
