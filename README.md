# Raksha's – Data Analytics Portfolio

Welcome to my data analytics portfolio. I use SQL, numerous Visualization tools,Power BI is one of them and Python to solve business problems, explore data, and build clear, actionable insights.

## About this repository

This repository showcases selected projects and practice case studies that demonstrate:
- Translating business questions into data questions
- Cleaning, transforming, and modeling data
- Creating dashboards and reports for non‑technical stakeholders
- Writing efficient, readable code and queries

Projects are organized by primary tool: **SQL**, **Power BI**, and **Python**.

## Repository structure

- SQL – SQL case studies and practice problems, including:
- Joins, subqueries, CTEs, window functions, aggregations
- Real‑world scenarios such as HR analytics, sales performance, and operations
  
- PowerBI – Interactive dashboards and reports, including:
- Data modeling, DAX measures, and calculated columns
- KPI tracking, drill‑through pages, and report design focused on storytelling

- Python – Data analysis and automation scripts, including:
- Data cleaning and EDA using pandas
- Simple visualizations and exploratory notebooks for analytics use cases

Each subfolder may contain its own `README.md` with more details about the individual project(s).

## How to navigate

1. Start with the folder that matches the skill you want to review (SQL, Power BI, or Python).
2. Open the project folder or file that interests you.
3. Check any project‑level `README.md` for:
   - Problem statement / business question  
   - Dataset description and assumptions  
   - Approach, key techniques, and sample outputs (queries, screenshots, or charts)  
   - Main insights or outcomes

## Skills and tools

- **SQL:** SQL Server / T‑SQL concepts, joins, CTEs, window functions, aggregations
- **Power BI:** Data modeling, DAX, measures, calculated columns, interactive dashboards
- **Python:** pandas, basic data wrangling and exploratory data analysis
- **Other:** Excel, basic statistics, data storytelling and documentation
